# Analisis-movilidad-economia-latam
Proyecto de análisis de datos que evalúa la relación entre la movilidad urbana (TomTom Traffic Index) y la productividad económica (OECD Cities) en las principales ciudades de Latinoamérica.

# Movilidad Urbana vs. Productividad Económica en LATAM 2024

## Objetivo del Proyecto
Analizar cómo la congestión vehicular afecta la economía de las principales metrópolis de Latinoamérica para identificar ciudades prioritarias para la inversión en infraestructura.

## Habilidades Técnicas Aplicadas
* **Limpieza de Datos:** Estandarización de formatos numéricos (cambio de comas por puntos) y tipos de datos con Pandas.
* **Análisis Exploratorio (EDA):** Filtrado de datos históricos y actuales (2024).
* **Fusión de Datasets:** Combinación de fuentes externas (TomTom y OECD) mediante llaves comunes.

## Hallazgos Principales
* **Bogotá:** Identificada como ciudad crítica debido a su alta congestión a pesar de tener un PIB per cápita menor que ciudades como Curitiba.
* **Recomendación:** Inversión urgente en infraestructura masiva y rediseño de calles para reducir cuellos de botella.

## Herramientas
* Python (Pandas, Numpy)
* Jupyter Notebooks
